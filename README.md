# 🚀 Clinical Trial Data Pipeline & Visualization (PySpark + Spotfire)

## 📌 Overview
This project demonstrates an end-to-end clinical trial data pipeline using PySpark, aligned with CDISC standards (SDTM & ADaM), and visualized using TIBCO Spotfire.

---

## 🛠 Tech Stack
- PySpark (Data Processing)
- Python
- Spotfire (Dashboarding)
- CDISC Standards (SDTM, ADaM)

---

## 📊 Features
- SDTM-like dataset ingestion  
- ADaM dataset creation  
- Data transformation and validation  
- Clinical insights dashboard using Spotfire  

---

## 📂 Project Structure
clinical-data-pipeline/

data/ → Raw and processed datasets  
notebooks/ → PySpark scripts  
spotfire/ → Dashboard and screenshots  
requirements.txt → Dependencies

## 🚀 How to Run

1. Install dependencies  
pip install -r requirements.txt  

2. Run scripts  
python notebooks/01_data_ingestion.py  
python notebooks/02_data_transformation.py  
python notebooks/03_adam_dataset_creation.py  
python notebooks/04_validation_checks.py
